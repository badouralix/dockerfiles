oh-my-via
=========

This **Dockerfile** builds an image with zsh, oh-my-zsh and [oh-my-via theme](https://github.com/badouralix/oh-my-via), for testing purpose.

## Usage

`docker run -it --rm badouralix/oh-my-via`
